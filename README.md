# Физический рисунок. v1.0
Набор команд, стилей и примеров для красивых физических рисунков в LaTeX

## Начало работы

Скачайте `design.sty` в директорию с вашим `.tex` файлом и добавьте его в преамбуле:

```
% in preamble
\usepackage{design}
```
